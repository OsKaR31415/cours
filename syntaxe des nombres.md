# Nombres en Ocaml

## les int
Les `int` sont simplement écris

```ocaml
# 1+1;;
- : int = 2

# 2*3;;
- : int = 6

# 1/2;;
- : float = 1.5

```

## les float
Les `float` sont avec des `.`
Les opérateurs sur des `float` doivent être aussi suivis par un `.` (ex: `+.`, `*.` etc.)

```
# 1.
- : float = 1.0

# 1. +. 3.5
- : float = 4.5
```
