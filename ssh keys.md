# Getting the keys
 - **Private :** `cat ~/.ssh/id_rsa` (sometimes `id_dsa`)
 - **Public :** `cat -/.ssh/id_rsa.pub` (sometimes `id_dsa.pub`)

# Generating a pair of keys

 - in the `~/.ssh` directory
 - using the `ssh-keygen -o` command
