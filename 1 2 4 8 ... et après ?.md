## Somme de tous les chiffres précédents

**1, 1, 2, 4, 8, 16, 23,** 28, 38, 49, 62, 70, 77, 91, 101, 103, 107, 115, 122, 127, 137 ...

| nombre   | somme de tous les chiffres |
| -------- | -------------------------- |
| $1$      | $1$                        |
| $1$      | $1+1 = 2$                  |
| $2$      | $1+1+2=4$                  |
| $4$      | $1+1+2+4=8$                |
| $8$      | $1+1+2+4+8=16$             |
| $16$     | $1+1+2+4+8+1+6=23$         |
| $23$     | $1+1+2+4+8+1+6+2+3=28$     |
| $28$     | $23+2+8 = 38$              |
| $38$     | $38+3+8 = 49$              |
| $\vdots$ | $\vdots$                   |
