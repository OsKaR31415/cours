# git configuration

```bash
$> git config --global user.name "Thierry Brouard"  # nom d'utilisateur local
$> git config --global user.email brouard@univ-tours.fr  # email de l'utilisateur
$> git config --global core.editor vim  # éditeur utilisé
```
