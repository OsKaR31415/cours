# associativité
Une loi de comlosition interne $*$ sur un ensemble $E$ est associative ssi : $\forall(a,b,c)\in E^3, a*(b*c) = (a*b)*c$
Si la loi est associative, on peut écrire : $a*b*c$ (on a plus besoin des parenthèses)
