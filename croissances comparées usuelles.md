# croissances comparées usuelles
[[croissances comparées]]

 - $e^x$
 - $x^{k}$ avec $k\in \R^{+*}$
 - $\ln^{k}(x)$ avec $k\in\R^{+*}$
     - aussi vrai pour les $\ln^{k}(x^{j}), j\in\R^{+*}$ car $\ln^{k}(x^{j}) = j\ln^{k}(x)$
 - $x^{k}\ln(x)$ avec $k\in\R^{+*}$
